Video

This is still beta. Video Controls are implemented using the awesome library videojs. Data should be pack as a json object with a property 'video'. This property in turn should be a two dimensional array with the objects to draw in the canvas. Each element of this array have 5 parameters. Please see the source code in this page to learn by example. The last parameter of each array is an array containing all the info used by the functions drawText, drawLine and drawShape in this library (see the documentation page at the end). BTW, this page is not functional in Internet Explorer).
