EXT-JS


Usage


Please see the javascripts libraries included this page which should look something like:

Flash fallback for IE 


              <link type="text/css" src="./js/EXT/resources/css/ext-all.css"></script>
              <link type="text/css" src="./css/color-field.css"></script>
              <!--[if lt IE 9]><script type="text/javascript" src="./js/flashcanvas.js"></script><![endif]-->
              <script type="text/javascript" src="./js/EXT/adapter/ext/ext-base.js"></script>
              <script type="text/javascript" src="./js/EXT/ext-all-debug.js"></script>
              <script type="text/javascript" src="./js/canvasXpress.min.js"></script>
              <script type="text/javascript" src="./js/color-field.js"></script>
              <script type="text/javascript" src="./js/ext-canvasXpress.js"></script>

or

VML fallback for IE 


              <link type="text/css" src="./js/EXT/resources/css/ext-all.css"></script>
              <link type="text/css" src="./css/color-field.css"></script>
              <!--[if IE]><script type="text/javascript" src="./js/excanvas.js"></script><![endif]-->
              <!--[if IE]><script type="text/javascript" src="./js/extext.js"></script><![endif]-->
              <script type="text/javascript" src="./js/EXT/adapter/ext/ext-base.js"></script>
              <script type="text/javascript" src="./js/EXT/ext-all-debug.js"></script>
              <script type="text/javascript" src="./js/canvasXpress.min.js"></script>
              <script type="text/javascript" src="./js/color-field.js"></script>
              <script type="text/javascript" src="./js/ext-canvasXpress.js"></script>

Make sure when you are using FlashCanvas to include the flash binaries in the same directory where the JavaScript file is located. Please see the documentation.


Right click anywhere in the graph to modify parameters
