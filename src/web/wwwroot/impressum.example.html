How to change this page

  1. Create a file impressum.html or copy impressum.example.html anywhere in the filesystem of the webserver serving TBro.
  2. Add content as you wish.
    • Static html will be displayed, php will not be processed.
    • Do not inclued html, head and body tags for the file will be included as part of the body of an existing html file (with navigation bar, etc).
    • Javascript is working.
    • You can take advantage of Foundation 4 style classes.
  3. Then set the value of IMPRESSUM_FILE in your TBro config.php to the full path to this file.
  4. Make sure, that your webserver user (eg www-data) has read access to this file.
If you are running TBro as a Docker container you can do the following as root on your host machine after creating impressum.html: 
cp impressum.html /var/lib/docker/aufs/mnt/$(docker inspect -f '{{.Id}}' TBro_official)/var/www/html/
docker exec -i -t TBro_official bash -c "sed -i 's/impressum.example.html/impressum.html/' /etc/tbro/config.php"
